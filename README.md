## Tools

### CVE Lookup Tool
Simple Python script to query the National Vulnerability Database (NVD) for CVE details.

**Usage:**
```bash
python cve_lookup.py CVE-2024-1234
```

**Features:**
- Fetches CVE details from NVD API 2.0
- Displays vulnerability description, severity, CVSS scores
- Shows affected products and references
- No API key required for basic queries
